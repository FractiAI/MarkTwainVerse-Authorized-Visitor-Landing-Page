## ğŸ—ï¸ Entity Types

### Building
Structures that define your world's architecture.
- **Examples**: Town halls, houses, monuments, shops
- **Behaviors**: Pulse with activity, glow at night, expand/contract
- **Energy**: Usually 0.6-0.9

### Character
Living beings that interact and move.
- **Examples**: Hero hosts, wanderers, shopkeepers, animals
- **Behaviors**: Greet visitors, tell stories, wander, sleep at night
- **Energy**: Usually 0.7-0.95

### Landscape
Natural environments and ecosystems.
- **Examples**: Forests, mountains, lakes, skies
- **Behaviors**: Seasonal changes, weather effects, wildlife sounds
- **Energy**: Usually 0.8-1.0 (highest natural energy)

### System
Abstract mechanisms that create dynamics.
- **Examples**: Economic systems, weather patterns, social networks
- **Behaviors**: Fluctuate, cascade effects, create emergent patterns
- **Energy**: Variable 0.5-0.9

## ğŸŒŸ Best Practices

### 1. Start Simple
- Begin with 5-10 entities
- Enable only essential cycles
- Add 2-3 key events
- Test before adding complexity

### 2. Use AI Assistance
- Describe what you want in natural language
- Let AI suggest configurations
- Iterate and refine suggestions
- Learn from AI examples

### 3. Balance Energy
- Keep average world energy 0.7-0.9
- Distribute energy across entity types
- High energy = active, low = dormant
- Use energy for dynamic behaviors

### 4. Test Performance
- Preview frequently during build
- Monitor frame rate (target 60fps)
- Reduce complexity if sluggish
- Use performance tab in browser DevTools

### 5. Leverage Templates
- Start with existing templates
- Modify incrementally
- Learn from MarkTwainVerse structure
- Build library of reusable patterns
